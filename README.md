# witness

> *what do you want to say but think no one needs to hear?*

**[witness-d2cm.onrender.com](https://witness-d2cm.onrender.com)**

---

## what is this

A place to say what you've been holding inside — anonymously, no account, no judgment.

Someone else is reading. A real person. They can press one button — **"i heard you"**.

That's it. No likes, no profiles, no algorithm. Messages disappear after 24 hours.

---

## why

Twitter is a performance. Instagram is a mask.

But some things you just want to say — not for attention, not for reactions. Just to be heard.

This is for those moments.

---

## how it works

- write one sentence — or more
- everyone sees it in real time
- press "i heard you" if something resonates
- in 24 hours it's gone forever
- no one knows who you are

---

## stack

- **Node.js** — server
- **WebSocket** — real-time connection
- **HTML / CSS / JS** — frontend with no frameworks
- **Render** — hosting

---

## run locally

```bash
git clone https://github.com/Azenko01/witness.git
cd witness
npm install
npm start
```

Open `http://localhost:3000`
